## 📋 Cambios Realizados

Se realizaron 1 cambio(s):

1. Fixed 'proxy_set_header X-Forwarded-Proto https' to 'proxy_set_header X-Forwarded-Proto $scheme': The server listens on port 80 (HTTP only), but X-Forwarded-Proto was hardcoded to 'https', which incorrectly tells the backend that all requests arrived via HTTPS. Using $scheme accurately reflects the actual protocol (http or https) used by the client.
