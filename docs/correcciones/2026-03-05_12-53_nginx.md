## 📋 Cambios Realizados

Se realizaron 2 cambio(s):

1. Added 'fastcgi_param SCRIPT_FILENAME $request_filename;' inside the nested PHP location block under '/app' (alias location). When using 'alias', the http-level 'fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name' produces an incorrect path (e.g., /var/www/html/app/index.php instead of /var/www/application/index.php). Using $request_filename correctly resolves the alias.
2. Added 'fastcgi_param SCRIPT_FILENAME $request_filename;' inside the nested PHP location block under '/blog' (alias location). Same issue as /app — alias requires $request_filename to compute the correct script path (e.g., /var/www/blog/index.php instead of /var/www/html/blog/index.php).
